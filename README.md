# Azure-End-End-Project
This Project helps in understanding how various azure services &amp; tools are used to complete end-end data projects.
In this project we will learn:
1. How to move on-premises data to the cloud using Azure Data Factory (ADF) with a Self-Hosted Integration Runtime.
2. How to integrate data factory with Git for version control and enterprise-grade development.
3. How to use Azure Data Lake Storage Gen2 as a central data repository.
4. How to improve reliability with Logic Apps for email alerts and Azure Monitor for detailed tracking.
5. How to perform data cleaning and transformation using Azure Databricks.
6. How to implement security best practices with Azure Key Vault + Databricks Secret Scope to protect sensitive secrets.
7. How to analyze data with Azure Synapse Analytics using SQL.
8. How to build interactive dashboards in Power BI for business insights.
